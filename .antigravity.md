# Harness Landing - Antigravity AI Instructions

## Project Overview
Reusable Next.js 15 landing page template for education, events, and campaigns. Designed for non-developers to customize with AI tools.

## Tech Stack
- Next.js 15 (App Router) + React 19 + TypeScript strict
- Tailwind CSS v4 (CSS-first configuration)
- Framer Motion (scroll animations)
- Lucide React (icons)
- Pretendard font (Korean-friendly)

## Mission Control Quick Reference

### Content Customization
All site text is centralized in `src/lib/data.ts`. To change any text on the site, edit this single file. Objects include:
- `siteConfig` - Global metadata
- `navLinks` - Navigation items
- `heroData` - Hero section content
- `featuresData` - Feature card content
- `galleryData` - Gallery items
- `ctaData` - Call-to-action section
- `footerData` - Footer content

### Brand Color Customization
Edit `--color-brand-*` CSS custom properties in `src/app/globals.css` under the `@theme` block.

### Adding Sections
1. Create a new component in `src/components/sections/`
2. Define content data in `src/lib/data.ts`
3. Import into `src/app/page.tsx`
4. Use `AnimatedSection` wrapper for scroll animations

### Key Components
- `Button` - Animated button (variants: primary, secondary, outline, ghost)
- `Card` - Flexible card with hover animation
- `AnimatedSection` - Scroll-triggered fade-in wrapper
- `Navbar` - Responsive with mobile hamburger menu
- `Footer` - Simple footer with links

### Commands
- `npm run dev` - Start dev server at localhost:3000
- `npm run build` - Production build
- `npm run lint` - ESLint check

### Guidelines
- Keep changes simple and minimal
- All text in `data.ts`, never in component files
- Use Tailwind utility classes for styling
- Images go in `public/` directory

## Deployment: GitHub → Vercel
GitHub push → Vercel 자동 재배포. `.env.example` 키를 Vercel 환경변수에 입력.

## SEO 설정
- `src/app/sitemap.ts` — sitemap.xml 자동 생성
- `src/app/robots.ts` — robots.txt 자동 생성
- `.env.local`의 `GOOGLE_SITE_VERIFICATION` → Google Search Console 인증
- `.env.local`의 `NAVER_SITE_VERIFICATION` → 네이버 서치어드바이저 인증
- `public/og-image.png` (1200×630px) 추가 시 카카오톡/SNS 공유 미리보기 활성화

## 도메인 연결 (가비아 → Cloudflare → Vercel)
1. 가비아 도메인 구매 → Cloudflare Add a Site → 네임서버 복사
2. 가비아 네임서버 변경 → Vercel Domains에 도메인 추가
3. Cloudflare DNS: CNAME `@` + `www` → Vercel CNAME (Proxy: DNS only)
4. `NEXT_PUBLIC_SITE_URL` 실제 도메인으로 변경 후 재배포
